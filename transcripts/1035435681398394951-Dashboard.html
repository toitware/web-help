floitsch 10/28/2022 10:00 AM
We are migrating away from Toitware hosted services, towards a model "v2".
The old v1 service is still accessible under auth.toit.io, but we recommend to start using our Toit v2 alpha instead.
The v2 Toit code is now open source, and can be adapted for similar use cases as our v1, and many of our customers do exactly that.
However, we are also working on a tool to manage fleets from the command line. Contrary to v1, the broker would not be owned by us, but would be a Supabase or MQTT server owned by the client.
