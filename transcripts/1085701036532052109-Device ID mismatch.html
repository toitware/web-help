bitphlipphar 03/16/2023 04:12 AM
Typically rescanning is enough to fix this (jag scan), but we try to do that automatically. The ID is random and put into the firmware to make sure Jaguar knows which device it is talking to. A failed firmware update could get you in a state where Jaguar believes that the device has been updated to a new ID, but the device has rolled back or rejected the update.
